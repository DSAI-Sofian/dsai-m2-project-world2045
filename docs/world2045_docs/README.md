# World2045 Platform

World2045 is a country-year analytical platform designed to analyze long-term development trajectories from 1950–2045.

The platform integrates multiple global datasets including population, economic indicators, inequality metrics, governance indicators, climate vulnerability, conflict data, and human development indices.

Architecture:

Bronze → Raw ingestion  
Silver → Cleaned country-year fact tables  
Gold → Analytical feature mart  
Model Layer → Forecast-ready dataset

Key analytical outputs include:

- Development trajectories
- Inequality concentration
- Governance dynamics
- Climate vulnerability
- Conflict exposure

The final modeling panel contains:

159 countries  
96 years (1950–2045)  
15,264 rows
